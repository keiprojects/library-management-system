Library Management System

Documentation + Defense Presentation Page

This page is prepared for student capstone/thesis defense to explain the system scope, architecture, user flow, and quick demo runbook.

1) Setup 2) Demo Login 3) Borrow & Return 4) Reports 5) Q&A

Project Overview

  • Role-based system with Admin and Borrower accounts.
  • Tracks books, borrowers, borrow records, returns, and penalties.
  • Supports borrower registration with student ID upload and approval workflow.

Tech Stack

  • PHP 8+, MySQL 8+
  • Tailwind CSS (CDN), Vanilla JavaScript
  • XAMPP-friendly deployment for school labs

How to Start (Quick Guide)

  • Start Apache and MySQL in XAMPP.
  • Create DB: library_management_system.
  • Import database/schema.sql, then database/seed.sql.
  • Open http://localhost/library-management-system/login.php.
