# Documentation

This folder contains project documentation for setup, usage, and student defense presentation.

## Contents

- `GETTING_STARTED.md` - complete setup and run instructions
- `DEFENSE_GUIDE.md` - suggested flow and script for capstone/thesis defense
- `defense-presentation.html` - one-page visual presentation used during defense

## Suggested Usage

1. Read `GETTING_STARTED.md` to prepare the project environment.
2. Practice with `DEFENSE_GUIDE.md` before final presentation day.
3. Open `defense-presentation.html` in browser and present section-by-section.
